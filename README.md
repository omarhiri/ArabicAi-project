# Arabic AI Hub

موقع حديث باللغة العربية لتجميع موارد الذكاء الاصطناعي من بنكات جاهزة، مسارات تدريبية مجانية، عروض مدفوعة، ومساحات مخصصة للإعلانات وروابط العمولة.

## البدء
1. افتح `index.html` مباشرة في المتصفح لمعاينة التصميم.
2. تم بناء التصميم باستخدام HTML و CSS فقط مع خطوط Google (Cairo و Space Grotesk).
3. الأقسام تشمل:
   - بطل الصفحة (Hero) مع بنكة جاهزة.
   - مسارات تدريبية مجانية ومدفوعة.
   - مكتبة بنكات مصنفة.
   - عروض ومنتجات مدفوعة ومراجعات عمولة.
   - مساحة Adsense وأقسام ثقة المجتمع والنشرة الأسبوعية.

يمكنك تخصيص النصوص، إضافة كود إعلانات Adsense، أو ربط النماذج بأدوات التسويق المفضلة لديك.
